## 一、世界观核心设定

### 1.1 开场场景

**黎恩·舒华泽**（闪之轨迹主角）在一场激烈战斗后（与巨大影牙兽的战斗），鬼之力暴走，昏迷倒地。醒来时，发现自己躺在一间温暖的森林守护者小屋——林间空地（The Glade）的守护者居所。

**醒来时的五感**：
- **视觉**：石砌壁炉的橙色火光，昏暗中隐约可见草药悬挂、木架上的短剑
- **听觉**：屋外的风声、远处偶尔传来的兽嚎、屋内的噼啪声
- **嗅觉**：草药、松脂、篝火的气味
- **触觉**：身上盖着柔软的兽皮，伤口被人处理过
- **人物**：一位粉发琥珀色瞳的女性（精灵族，外貌与人类一致无尖耳）——Seraphina，正在照看他

**关键设定**：
- 黎恩失去了部分记忆，不记得自己是如何来到这片森林的
- 但他保留了所有战斗能力、八叶一刀流剑术、鬼之力，以及与瓦利玛的契约
- Seraphina 是 Eldoria 森林最后的守护者，对这位外来者既警惕又好奇
- 这片森林正在被腐化——影牙兽（Shadowfangs）不断从腐化的土地中诞生
- 黎恩的鬼之力与森林的腐化似乎存在某种共鸣……

### 1.2.1 新增角色：玲·布莱特

**身份**：前噬身之蛇执行者No.XV「杀戮之天使」，导力天才少女，外表天真烂漫的哥德萝莉小恶魔，实则危险至极。粉发红瞳，手持巨大镰刀型导力武器。

**三线定位**：
- **纯爱路线**：危险但逐渐被感化的小恶魔，从捉弄黎恩到真心祝福，最终在两人的守护下找到归处
- **NTRS路线**：最不可预测的参与者，以"游戏"心态介入，既是观察者又是催化剂，带来独特的紧张感和意外转折
- **被动NTR**：敏锐的观察者，会用残忍戏谑的方式点破黎恩不愿面对的真相，加速关系的崩塌或转变

### 2.1 三条路线的定义与区别


### 2.2 各路线触发条件

| 路线 | 触发条件（核心变量阈值 + 事件链） |
|---|---|
| **纯爱** | trust_level ≥ 60, bond_level ≥ 40, ntrs_awakened = 0, abandonment_count < 30, seraphina_despair < 50, 玩家选择守护路线 |
| **NTRS** | trust_level ≥ 50, bond_level ≥ 30, ntrs_awakened = 100, seraphina_acceptance ≥ 40, abandonment_count < 40, seraphina_despair < 60 |
| **被动NTR** | abandonment_count ≥ 40 + seraphina_despair ≥ 60 + trust_level < 40, ntrs_awakened 可以是 0 或 100（不影响触发） |

## 三、三线共存的设计哲学

### 3.1 三条路线的情感张力核心

| 路线 | 核心情感张力 | 关键场景 |
|---|---|---|
| **纯爱** | "我愿意用生命守护你，你是我唯一的光" | 并肩战斗、深夜疗伤、告白时刻 |
| **NTRS** | "我想看到你被他人欲望，因为证明你的魅力，也因为我变态的共享欲；但请记住，你最终属于我" | 坦白XP的对话、约定边界、共享时刻后的占有欲爆发、重归于彼此的仪式 |
| **被动NTR** | "我没有保护好你，我一次次不在你身边……现在我只能看着你被夺走。但我还有力量，我会重新赢得你，或者……接受这一切" | 保护失败的战斗、缺席的关键时刻、见证场景、觉醒与夺回 |

### 3.2 为什么三条路线都能成立

1. **黎恩的心理基础**：他的核心心理是"保护欲 + 自毁倾向 + 自卑"。这三者的组合在不同情境下会走向不同方向：
   - 保护欲被满足 → 纯爱
   - 保护欲与隐藏的"见证欲/共享欲"混合 → NTRS
   - 保护欲失败、自毁倾向主导 → 被动NTR

2. **Seraphina的心理基础**：她的核心是"孤独守护者 + 炽天使血脉的神圣/压抑二元性"：
   - 找到能平等守护彼此的人 → 纯爱
   - 发现恋人有特殊XP，理解并愿意尝试 → NTRS
   - 连续失望后产生动摇 → 被动NTR

3. **两人都是强者**：没有任何外力能真正强迫他们。被动NTR的"被夺走"，本质上是"在连续失败的心理状态下，暂时失去了主动夺回的意愿"——而这种意愿随时可以重新点燃。

## 五、核心设计原则（编写所有内容时都要遵循）

1. **黎恩永远是黎恩**：即使在被动NTR最黑暗的时刻，他仍然保留着"灰色骑士"的内核——温柔、保护欲、责任感。他可以暂时失去斗志，但不会变成另一个人。

2. **Seraphina永远是Seraphina**：精灵守护者的优雅、神圣与孤独感贯穿所有路线。即使在被"夺走"时，她对黎恩的情感也永远是"复杂的混合"，而非简单的背叛。

3. **NTRS的核心是"共享"而非"背叛"**：这是一对恋人在探索彼此的特殊欲望，双方都保持对对方的爱与最终控制权。区别于传统NTR的痛苦和背叛感。

4. **被动NTR的核心是"累积失败"**：不是一次事件，而是多次黎恩不在场/失败后的累积结果。但因为两人都是强者，这种"失败"也意味着随时可以逆转。

5. **路线转换是自由的**：NTRS可以回归纯爱（决定"只属于彼此"），被动NTR可以转为NTRS（在痛苦中发现这是一种扭曲的共享欲）或纯爱（重新赢得）。

6. **对话内容的分层设计**：世界书中的 content 使用纯文字描述区分三条路线——通过文字说明"如果处于纯爱路线则XX，如果处于NTRS路线则YY"的方式来表达不同路线的内容差异。不使用条件判断语法（世界书不支持）。

7. **变量驱动一切**：所有行为差异、对话内容差异、事件触发差异，全部由 03_变量系统.md 中定义的变量值驱动。
