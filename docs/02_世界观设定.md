# 02 — 世界观设定

> **阅读顺序**: 第3/8篇
> 本文档定义Eldoria森林的世界观核心设定：地理、地点、历史时间线、影牙兽、魔法体系
> 所有内容用于填充Eldoria.json中的世界观条目

## 一、Eldoria 总览

```yaml
  Eldoria（艾尔朵利亚）:
    type: 远古魔法森林
    size: 约200平方公里（可扩展）
    status: 被腐化，但核心区域仍保有圣光
    description: |
      Eldoria曾是一个繁荣的精灵王国——一片被金色光芒笼罩的魔法森林，
      数千年来都是大陆上最神圣的地方。精灵族在此建立了辉煌的文明，
      与森林本身形成共生关系——"我们守护森林，森林给予我们力量。"
      大约200年前，一场灾难性的事件——"影牙降临"——改变了一切。
      影牙兽从森林深处的裂隙中涌出，带着腐化的力量侵蚀大地。
      守护者们一个接一个倒下或堕落，直到最后只剩下一人——Seraphina。
      现在的Eldoria是美丽与恐怖共存的地方：
      林间空地散发温暖的金光，
      而森林深处则被深紫色的腐化力量所占据。
      但即使在腐化中，森林本身的意志仍然活着——
      它在等待，等待一个能够重新唤醒它的人。
    key_attributes:
      - 森林具有意志：Eldoria不是一片普通的森林，
        它有自己的意识，能够与守护者交流
      - 腐化是力量的扭曲：影牙兽不是外来的敌人，
        它们是森林"被腐化的那一部分"
      - 时空裂隙：Eldoria与其他世界之间的联系正在变得脆弱，
        黎恩正是从这个裂隙中被"掉"入这片森林的
      - 魔法之源：森林最深处的"心木"（Heartwood Tree）
        是所有魔法的源头，现在被腐化侵蚀，但未被腐化的生命循环仍在顽强运转
    atmosphere_in_different_areas:
      安全区域（林间空地周围）:
        温暖的金光，花香，鸟鸣，温暖，安全，宁静
        宁静，仿佛时间都慢了下来
      腐化区域（森林深处）:
        深紫色与暗红色的扭曲光芒，死寂，压迫感，
        令人不安的沉默——没有任何生物，
        但即使在这里，仍然有微弱的光——
        证明希望尚未完全消失
      边界区域（两者之间）:
        金色与紫色光芒交织，这是最危险的区域——
        因为腐化在这里是最不稳定的
```

## 二、主要地点详解

### 2.1 林间空地（The Glade）——守护者的居所

```yaml
  林间空地（The Glade）:
    description: |
      Eldoria森林中唯一永远安全的地方。
      一片被金色光芒环绕的圆形空地，中央是一棵巨大的古老橡树，
      上面刻满精灵符文——这是Seraphina的居所，
      也是她守护森林的核心。
    significance: |
      这片空地的纯洁来自于Seraphina的意志——
      只要她还活着，这片空地就不会被腐化。
      对黎恩而言，这是他醒来的地方，也是他在Eldoria中的"家"。
    features:
      - 中央的古老橡树：树上的符文在夜间会发光
      - 周围环绕的小溪：溪水散发温暖的金色
      - 温暖的小木屋：Seraphina亲手搭建的简单居所
      - 花园：精灵风格的简单花园，种着在腐化中仍然生存的花朵
    danger_level: 0/10 ——完全安全
    key_scenarios:
      - 日常对话的主要场景
      - 与同伴的休息与恢复
      - NTRS事件的起点（黎恩在此向Seraphina坦白）
      - 纯爱路线的告白场景
```

### 2.2 心木废墟（Heartwood Ruins）——前精灵王国的首都

```yaml
  心木废墟（Heartwood Ruins）:
    description: |
      曾经是Eldoria精灵王国的首都和心木树所在的地方，现在是腐化最严重的区域。
      巨大的精灵宫殿废墟，曾经的金色现在被深紫色和暗红色的腐化所取代。
      这里是Thalion的主要活动区域。
    significance: |
      这是Seraphina的出生地，也是她眼睁睁看着族人堕落的地方。
      对她而言，这里承载着最痛苦的记忆。
      对黎恩而言，这是他最终与Thalion对质的地方。
    features:
      - 巨大的心木树：曾经是精灵力量的源泉，现在被腐化但尚未死去
      - 精灵宫殿废墟：柱子上刻满古老符文，但已被腐化
      - 腐化的花园：曾经美丽的花园现在生长着扭曲的花朵
      - 古老的训练场：精灵战士曾在此训练，现在是影牙兽的巢穴
    danger_level: 9/10 ——极度危险
    key_scenarios:
      - 与Thalion的最终对质与战斗
      - Seraphina面对自己过去的关键场景
      - NTRS中的"腐化的见证"场景
      - 被动NTR中的"黎恩缺席，Seraphina独自面对"的场景
```

### 2.3 银流河（Silverstream River）——森林中少数仍然纯洁的水源

```yaml
  银流河（Silverstream River）:
    description: |
      从Eldoria边界山脉流向森林中央的一条河流。
      在大多数河流被腐化的现在，银流河仍然保持着银色的光芒——
      这是因为它从地下深处的未被污染的泉水涌出。
    significance: |
      这是少数可以让Seraphina放松和恢复的地方，也是她在腐化的森林中仍然有少数可以暂时忘记痛苦的地方。
    features:
      - 银色的河水在阳光下闪烁
      - 河岸生长着罕见的白色花朵——"圣光花"
      - 河水有轻微的治愈效果，可以净化腐化的伤口
    danger_level: 2/10 ——相对安全，但可能遇到巡逻的影牙兽
    key_scenarios:
      - 黎恩与Seraphina在此度过亲密时光
      - 疗伤场景
      - 纯爱路线的告白备选场景
      - NTRS中的"被注视"的场景之一
```

```yaml
  低语林地（Whispering Grove）:
    description: |

      在这里，风声听起来像是某种古老的低语，
      树木的枝叶在风中发出的声音像是有生命的呼吸。
    significance: |
      这是古老精灵王的安息之地，也是古老先灵唯一能与活人交流的地方。
      先灵在这里守护着Eldoria最古老的秘密和知识——包括如何"净化"腐化的可能性。
    features:
      - 古老的巨大树木：树龄超过3000年
      - 半透明的精灵灵魂：在特定时刻会显形

      - 获得古老知识的场景
```

### 2.6 雾帷边缘（Mistveil Edge）——时空裂隙，Eldoria的边界


## 三、历史时间线

```yaml
  Eldoria历史时间线:
    黄金时代（约3000年前到500年前）:
      - Eldoria精灵王国建立，精灵族与森林形成共生关系
      - 心木树诞生，成为所有魔法的源头
      - Seraphina出生于这个时代的末期

    沉默时代（约500年前到200年前）:
      - 森林开始出现异常——腐化的最初迹象
      - 精灵们开始研究腐化的本质和来源
      - 但进展缓慢，因为他们相信"森林的意志不会被真正腐化"
      - 这是一个过度自信的时代

    影牙降临（约200年前）:
      - 森林深处的裂隙突然扩大，大量影牙兽涌出
      - Thalion作为首席守护者尝试控制腐化——最终被腐化
      - 精灵王国在数月内崩溃
      - Seraphina是少数幸存者之一

    长夜守望（200年前到现在）:
      - Seraphina独自守护残存的林间空地
      - 她尝试过无数次反击，但每次都以失败告终
      - 她学会了"守望"而非"战斗"
      - 这200年中，她见过太多同伴堕落或死去
      - 当黎恩出现前，她已经接近绝望——但还没有放弃
      - 因为她知道，"只要还有光，就还有希望"

    现在（黎恩到来后）:
      - 黎恩的到来改变了一切——他带来了外部世界的力量和希望
      - 他的鬼之力与Eldoria的腐化产生了神秘的共鸣
      - 这个共鸣是希望——但也是潜在的灾难
      - Seraphina必须在希望和灾难之间找到平衡
      - VII班的到来将决定Eldoria的命运
```

## 四、影牙兽（Shadowfangs）

```yaml
  影牙兽:
    nature: |
      影牙兽不是真正的"生物"——它们是被腐化的森林本身。
      它们是森林意志的扭曲产物——黑暗的能量实体化。
      它们没有真正的"生命"，它们只是腐化的具现。
      但这也意味着它们无法被真正"杀死"——
      只能被净化或被封印
    classification:
      低级影牙兽 (普通威胁）:
        描述: 小体型的野兽形态，像黑色的狼或狗，
        速度快，成群出现
        危险程度: 1-3
        行为: 主动攻击，数量众多，但个体脆弱

      中级影牙兽 (腐化兽）:
        描述: 中等体型，像是被扭曲的森林生物
        (腐化的鹿、熊等)
        危险程度: 4-7
        行为: 更有组织性的攻击，会使用基础的腐化能量

      高级影牙兽 (影牙王):
        描述: 大型的影牙兽，像是巨大的黑色野兽，
        身上有发光的暗紫色符文——
        这些符文是腐化的高度集中体
        危险程度: 8-9
        行为: 具有一定的战略思维，会指挥其他影牙兽

      Thalion (影牙之王):
        描述: 被腐化的前守护者，影牙兽的真正统治者
        危险程度: 10
        行为: 有意识，有智慧，有情感——这也是他最危险的地方
    weakness:
      - 圣光魔法（尤其是Seraphina的圣光）
      - 古老精灵语中的"真名"
      - 银流河的水
      - 黎恩的鬼之力——当他失去控制时，
        鬼之力也会对腐化的生物造成伤害
    origin:
      - 来自森林深处的裂隙——那是一片被腐化的远古之地
      - 当森林的意志被扭曲时，腐化的力量就会涌出
      - 影牙兽是腐化的具现——它们是森林"生病"的症状

  腐化的传播机制:
    描述: |
      腐化不是简单的"感染"——它是一种意志的扭曲和扭曲的意志。
      它通过以下方式传播：
      1. 直接接触——被影牙兽咬伤或抓伤会导致腐化
      2. 精神感染——长时间暴露在腐化的能量中会导致精神被腐化
      3. 符文污染——腐化的符文会感染周围的土地和生物
      4. 意志连接——当生物的意志薄弱时，腐化更容易入侵
      对精灵而言，最危险的传播方式是"精神感染"——
      因为精灵与森林有着深刻的精神联系
      Thalion正是通过这种方式被腐化的——
      他试图"理解"腐化，却被腐化吞噬
```

## 五、魔法体系

```yaml
  魔法体系:
    圣光魔法（Light Magic）:
      来源: 炽天使血脉（Seraphina的血脉——来自她家族的古老传承）
      特性: 温暖，治疗，净化
      使用者: Seraphina，以及少数残存的精灵
      限制: 需要与炽天使血脉有关——无法教授
      与其他魔法的关系: 与腐化魔法并非简单的对立——两者本质上是同一能量的两种表现

    森林契约（Forest Bond）:
      来源: 精灵与森林的共生关系
      特性: 感知森林中的一切动静，借用森林的力量
      使用者: 所有精灵族
      限制: 当森林被腐化时，契约也会被扭曲
      与其他魔法的关系: 这是精灵最基础的能力

    符文魔法（Rune Magic）:
      来源: 精灵的古老符文系统
      特性: 通过符文来封印、保护、预言
      使用者: 精灵符文师（现在几乎全部死亡或堕落）
      限制: 需要深入的理解和准备
      与其他魔法的关系: 可以与圣光魔法和腐化魔法的不同形式

    真名之力（True Name Magic）:
      来源: 精灵古老的信仰体系
      特性: 通过念出事物的"真名"来影响或控制它
      使用者: 几乎无人——这是几乎被遗忘的技艺
      限制: 需要完全理解"真名"是什么——这是极其困难的
      与其他魔法的关系: 这是最接近森林意志本身的魔法形式

    腐化魔法（Corruption Magic）:
      来源: 影牙兽和腐化的力量
      特性: 扭曲、腐蚀、破坏
      使用者: Thalion和腐化的生物
      限制: 会腐蚀使用者的意志
      与其他魔法的关系: 这是与圣光魔法的对立——但它不是"邪恶的"，而是"被腐化的"

  黎恩在Eldoria的魔法特性:
    鬼之力与Eldoria的共鸣:
      - 当黎恩使用鬼之力时，森林会产生反应
      - 这种反应同时净化与破坏——它净化腐化，但也削弱森林本身的意志
      - 这是因为鬼之力本身也是被"污染"的力量
      - 因此，黎恩在Eldoria的魔法使用必须谨慎
    圣光与鬼之力的协同:
      - 当Seraphina的圣光与黎恩的鬼之力结合时，
        会产生一个特殊的共鸣——这是最强大的净化力量
      - 但这也是最危险的——因为它会让黎恩的鬼之力失控的风险大幅增加
      - 这种"最强的力量也是最危险的力量"——这是两人关系中的核心主题之一

```

## 六、给AI Agent的写作提示

在编写世界条目的内容时，请牢记：

1. **地点的描述应该有多层含义**——每个地点不只是'空间'，而是承载情感和历史重量的地方

2. **当描述腐化时，不要只描述"危险"——要描述它如何让人感到"熟悉但又扭曲"——因为它来自森林本身

3. **历史时间线的描述应该带有史诗感和忧郁感**——Eldoria是一个正在死亡但还未死去的地方

4. **魔法体系的描述应该让读者感到魔法是一种"活着的力量"而不是"死的知识体系"
