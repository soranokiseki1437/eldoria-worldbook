## 典型修复模式

### 1. 反应移到起因后

**N01 坦白之夜:**
- 前: `[1] 她说"还有兴奋"时自己先愣住了` → `[2] 深夜火炉余烬。她靠在他肩上` (反应跑在场景前)
- 后: 场景设定在前，反应移到坦白对话之后

**N02 边界协商:**
- 前: `[1] 她写下"可以碰"三个字时停了一下` → `[2] 坦白之夜后数日` (动作在时间设定前)
- 后: 时间设定→动作→诉求

### 2. 破折号→逗号句号

- "不是不满足是还有别的" ← "不是不满足是还有别的——现在才知道"
- "声音第一次透着不确定" ← "声音第一次不确定——"
- "规则不是限制，是安全网" ← "规则不是限制——是安全网"

### 3. 同义重复合并

大量事件中相邻bullet说同一件事的合并为一句

### 4. 元叙事总结移除

删除情境中的"从第一次到咽下第一口"等递进总结表述
